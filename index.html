Diljit Dosanjh Concert Ticket Booking

Scenario: You need to build a booking system for the Diljit Dosanjh concert. The system allows users to book seats for the concert, divided into three sections: VIP, General, and Economy.

1. Display Sections:

Show three sections: VIP, General, and Economy.

2. Seat Layout:

Each section has a different seat layout:

  • VIP Section: 5x5 grid (25 seats)
  • General Section: 6x6 grid (36 seats)
  • Economy Section: 8x8 grid (64 seats)
3. Seat Status:

Seats should be marked as booked after a user selects them. A user can book a maximum of 5 seats.

4. Booking Confirmation:

After selecting seats (max 5), the system should confirm the booking and show the seat numbers. Once booked, seats should not be available for further booking.
